# Chrome Dino Game (C++)

## Keybinds

- Jump = UP/SPACE/A
- Duck = DOWN/B
- Restart = ENTER/START
- Exit = ESCAPE

## Missing features from the original (no plan to implement)

- System for seasonal themes
- Accessibility features (e.g. synthesized obstacle warning, slow game mode)
