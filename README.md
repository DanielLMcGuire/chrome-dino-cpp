# Chrome Dino Game (C++)

## Keybinds:

- Up = UP/SPACE
- Down = DOWN
- Restart = ENTER
- Exit = ESCAPE

## Bugs:

- When switching to dark, it will not render everything properly inverted. The dino, cacti, and pterodactyl are still visible.
